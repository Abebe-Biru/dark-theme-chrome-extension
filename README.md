# Modern Dark Theme Browser

A Chrome extension that provides a sleek dark mode browsing experience with website-specific controls.

## Key Features

- **Global Dark Mode**: Enable/disable dark mode across all websites
- **Website-Specific Controls**: Customize dark mode per website
- **Multiple Intensity Levels**: Light, Medium, or Deep Black themes
- **Keyboard Shortcut**: Toggle with `Ctrl+Shift+D`
- **Persistent Settings**: Remembers your preferences

## Installation

1. Clone/download this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select this folder

## Usage

### Popup Interface
- Toggle dark mode globally or for current website
- Force dark mode for stubborn websites

### Options Page
- Manage global settings
- Configure website-specific preferences
- Set default intensity levels
